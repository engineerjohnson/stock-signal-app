# 年量排序交易方法說明

> 根據實戰影片逐字說明整理，適用於當沖飆股神手 APP 盤中操作。

---

## 1. 排序方式對比

| 排序 | 說明 | 適合情境 |
|------|------|----------|
| **連次排序** | 依外/內盤連續 tick 次數排序 | 觀察個股連次強弱（正=做多在上方，負=做空在下方，需分兩端看） |
| **連量排序** | 依連次期間累積量排序 | 觀察方向性成交量動能 |
| **年量排序** ⭐ | 依單筆最新成交量（lastDeltaVol）降序 | 量最大的股票排最前，不論方向；可從頭到尾掃，不用分上下兩端 |

> 老師說：「以前我是連次排序，但是觀察股票太多，連次排序我就必須看上面跟下面。我現在直接看年量。」

---

## 2. 年量排序的兩大核心訊號

### 2.1 發動量（第一次大量）

**定義**：|連次| = 1（方向剛剛翻轉）且單筆成交量很大

**意義**：主力剛剛「發動」，是最佳進場時機

```
條件：
  |連次| = 1
  單筆成交量 >= 100 張
  今日量比 >= 1.5x
```

**操作**：
- 做多：發動量出現 + 方向朝上（連次=+1）→ 跟進做多
- 做空：發動量出現 + 方向朝下（連次=-1）→ 跟進做空

---

### 2.2 轉折量（連20次以上大量）

**定義**：|連次| >= 20 且連量達市場前30%門檻

**意義**：行情已連續走了超過100秒（20個5秒週期），能量快耗盡，準備轉折

```
條件：
  |連次| >= 20
  連量 >= 市場前30%門檻（動態計算）
```

**操作**：
- 連漲20次以上 → 觀察轉折，準備做空
- 連跌20次以上 → 觀察轉折，準備做多

> 老師說：「第一次大量是發動量；連20次然後又很大量，那就是準備的轉折量。」

---

## 3. 買盤捷徑 / 賣盤捷徑

### 3.1 買盤捷徑（外盤持續大量 → 高點風險）

**定義**：外盤（主動買方）連續 >= 10 個 tick 攻擊且連量大

```
條件：
  連次 >= +10（外盤連續10次以上）
  連量 >= 市場前30%門檻
```

**意義**：買盤能量即將耗盡，容易出現短線高點
- 已做多者：考慮獲利了結
- 做空者：等候進場機會

---

### 3.2 賣盤捷徑（內盤持續大量 → 低點支撐）

**定義**：內盤（主動賣方）連續 <= -10 個 tick 攻擊且連量大

```
條件：
  連次 <= -10（內盤連續10次以上）
  |連量| >= 市場前30%門檻
```

**意義**：賣盤能量即將耗盡，容易出現短線低點
- 已做空者：考慮回補（如勵志案例在 9:32 賣盤捷徑點回補）
- 做多者：等候反彈進場

> 老師舉例（勵志股）：「這個611塊呢是連續10次，連續50秒賣的力道比買的力道還要強。他已經128張也爆量了。這邊上下都框起來了，很容易是一個短線的賣盤間接點，後面會反彈。」

## 4. 盤中做多三種方法

### 方法一：找發動量進場

適合 **主動出擊型**

1. 打開 APP → 切換「發動量」Tab
2. 用**年量排序**（單筆量大的在前）
3. 看到 |連次|=1 + 大量 → 確認方向為外盤（連次=+1）
4. 第一筆量大且後續跟上 → 進場做多

---

### 方法二：連綠爆量抓轉折做多

適合 **逆勢抄底型**

1. 用年量排序，找連量大 + 連次為負（連跌中）的股票
2. 觀察是否出現「賣盤捷徑」（連次 <= -10 + 爆量）
3. 賣盤耗盡後的反彈點 → 做多

---

### 方法三：找主投/權證/標的向上

適合 **籌碼面確認型**

| 方法 | Tab | 說明 |
|------|-----|------|
| 找主投買進 | 主投買進 | 連漲≥2次 + 大量 + 量比高 |
| 找權證做多 | 權證做多 | 漲+2%~+9.5% + 月線多頭 + 連次≥1 |
| 找標的向上 | 飆風向上 | 連漲≥3次 + 有連量 + 今漲 |

---

## 5. 盤中做空方法

### 方法一：找弱勢股 + 第一次大量

1. 切換「弱勢」Tab → 找連跌且今日下跌的股票
2. 搭配年量排序，找第一筆大量剛出現的
3. 確認方向為內盤（連次=-1）→ 進場做空

---

### 方法二：連紅爆量抓轉折做空

1. 用年量排序，找連量大 + 連次為正（連漲中）
2. 觀察是否出現「買盤捷徑」（連次 >= +10 + 爆量）
3. 買盤耗盡後的轉折點 → 做空

---

### 方法三：找權證做空 / 隔日沖出貨股

```
典型隔日沖出貨模式（如679案例）：
  昨日：鎖漲停 → 被法人買到 → 成本被抬高
  今日：主力需拉高出貨 → 開盤奮力一拉
  特徵：早盤大量但搭不上去 + 出現在年量排序前列
  操作：等拉高出貨型態確立後做空（9:14發現，9:33賣盤捷徑確認，回補）
```

## 6. 實操流程（盤中標準流程）

```
08:50  開啟 APP，頁面保持開著累積連次/連量資料

09:00  開盤
  → 切「權證做多」Tab，往下滑更新名單
  → 切換「年量排序」（⇅ → 年量 ↓高→低）

09:00~09:30  盤中掃描
  發動量 Tab：看到 |連次|=1 + 大量 → 考慮進場
  轉折量 Tab：|連次|≥20 + 大量 → 準備反向
  買盤捷徑：外盤連10次 + 大量 → 高點注意
  賣盤捷徑：內盤連10次 + 大量 → 低點注意

13:30  收盤，資料自動儲存
```

## 7. 連量五顏色說明

| 狀態 | 含義 |
|------|------|
| 連量顯示**正數橘色** | 外盤累積量多（買方強勢） |
| 連量顯示**負數藍色** | 內盤累積量多（賣方強勢） |
| 連次顯示**正數紅框** | 外盤連次（連續外盤成交） |
| 連次顯示**負數綠框** | 內盤連次（連續內盤成交） |

> **台股慣例**：紅色=漲=外盤，綠色=跌=內盤（與國際股市相反）
